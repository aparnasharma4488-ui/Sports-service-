# Sports-service-
Containerize a Sports service (built with Rust) using industry best practices for Docker. The project includes writing multi-stage Dockerfiles to optimize image size and security, alongside  a Docker Compose configuration to orchestrate the application and its database dependencies. This ensures a consistent, portable development .
